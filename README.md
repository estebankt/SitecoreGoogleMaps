SitecoreGoogleMaps
==================

The Sitecore Google Maps Module is an integration of the Google Maps API with Sitecore.

 The Sitecore Google Maps Module is an integration of the Google Maps API with Sitecore. The module has been completely rewritten by the team at IE(http://www.ie.com.au/), integrating with the new Google Maps API for Sitecore Version 6.5 (verified working from Sitecore Version 6.2 and beyond). The modules main improvements have been, usability, improved feature set with greater integration into the Sitecore interface.

More information about the comparison of the new and old module can be found on IE’s Blog(http://ie.com.au/content-management-systems/sitecore-google-maps-module)
